## 各科复习大纲

### 📐 数学
**初一重点（约2周）：**
- 有理数运算、整式加减
- 一元一次方程、二元一次方程组
- 不等式与不等式组
- 平面直角坐标系
- 相交线与平行线、三角形基础

**初二重点（约2周）：**
- 全等三角形、轴对称
- 整式乘除与因式分解
- 分式、二次根式
- 勾股定理、平行四边形
- 一次函数、数据的分析

**初三预习（约1周）：**
- 一元二次方程（预习）
- 二次函数（预习）

### ⚛️ 物理
**初二重点（约2周）：**
- 机械运动、声现象
- 物态变化、光现象
- 透镜及其应用
- 质量与密度

**初二下重点（约2周）：**
- 力与运动、压强
- 浮力
- 功与机械能
- 简单机械

### 🧪 化学（初三新学科，提前预习）
- 化学用语（元素符号、化学式）
- 常见物质的性质
- 化学实验基本操作

### 📝 语文
**古诗文（贯穿全程）：**
- 初一必背篇目复习
- 初二必背篇目复习
- 每周背诵2-3篇

**现代文阅读：**
- 记叙文阅读技巧
- 说明文阅读技巧
- 议论文阅读入门

**作文：**
- 每周1篇作文训练
- 素材积累（名言、事例）

### 🇬🇧 英语
**词汇：**
- 初一词汇（约800词）复习
- 初二词汇（约600词）复习
- 每日30词，循环记忆

**语法：**
- 时态综合（一般现在/过去/将来、现在进行、现在完成）
- 被动语态
- 宾语从句、定语从句
- 条件状语从句

**题型训练：**
- 完形填空（每周4篇）
- 阅读理解（每周6篇）
- 书面表达（每周2篇）
