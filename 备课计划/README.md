# 备课计划与课件管理

## 使用方式

### 生成课件
```bash
hermes-lesson 数学 "二次函数的图像与性质"
hermes-lesson 物理 "浮力原理"
hermes-lesson 英语 "Unit 6 Grammar"
```

### 课件功能
- 🌐 HTML格式，浏览器直接打开
- 🔊 每页可点击「朗读本页」语音播放
- 📱 ← → 键盘翻页
- 📂 按学科分类存储

### 备课流程
1. 各科老师根据暑假计划提前备课
2. 用 `hermes-lesson` 生成互动课件
3. 课件存储在 `课件/学科/` 目录
4. 知识库同步到GitHub，随时可访问
